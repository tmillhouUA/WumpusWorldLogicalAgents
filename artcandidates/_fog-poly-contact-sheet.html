☁ Fog Polygon ☁

Generalized fog fill: blob CENTERS are rejection-sampled to fall within an arbitrary polygon (pentagon / diamond), but every blob is drawn as a full, uncropped circle — never clipped to the polygon boundary. That's what lets fog spill across a shared edge into a neighboring region instead of stopping at a hard seam ("balloons tied to points in a polygon", not "balloons confined by a polygonal wall"). Red outline = the TRUE polygon boundary, shown for reference only (not part of the rendered fog).
Use Reroll to resample all fog.
